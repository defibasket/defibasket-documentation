# IndexPool

## CreatePortfolio

Mints a new NFT and allocates deposited funds to this new NFT.

## DepositPortfolio

Adds more funds into an existing NFT and rearrange allocation inside the portfolio.

## EditPortfolio

Rearrange allocation inside the portfolio. You can't deposit funds into this function neither to withdraw.

## WithdrawPortfolio

Rearrange allocation and then withdraws.
